# web

JSP와 MySQL을 이용해서 만든 웹페이지 

## 기술 스택
- Backend: JSP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## 주요 기능
1. 사용자 관리
   - 로그인/로그아웃
   - 회원가입
   - 계정 정보 수정
   - 계정 삭제
   - 마이페이지

2. 게시판 기능
   - 게시글 작성
   - 게시글 검색
   - 게시글 조회

## 프로젝트 구조
```
webProject/
├── WEB-INF/
├── META-INF/
├── js/
├── fonts/
├── css/
├── *.jsp (각종 JSP 페이지)
└── main.java
```

## 설치 및 실행 방법
1. 웹 서버(Apache Tomcat 권장) 설치
2. MySQL 데이터베이스 설치 및 설정
3. 프로젝트를 웹 서버의 배포 디렉토리에 복사
4. 웹 서버 실행 후 접속
